# AwesomeGNN

This repository currently hosts the course-project materials for a DINO-WM study on representation and uncertainty in action-conditioned world models.

## Included Content

- `docs/PROJECT_README.md`: project overview and current status
- `docs/EXECUTION_LOG.md`: execution history, engineering notes, and experiment log
- `docs/COURSE_REPORT_DRAFT.md`: report draft
- `docs/COURSE_PROJECT_PLAN.md`: current execution plan
- `docs/EXPERIMENT_TRACKER.csv`: experiment table

## Current Project Status

Completed:

- full three-seed formal matrix on PointMaze
- partial larger-sample planning reevaluation

In progress:

- larger-sample planning reevaluation for the remaining seed-1 and seed-2 configurations
